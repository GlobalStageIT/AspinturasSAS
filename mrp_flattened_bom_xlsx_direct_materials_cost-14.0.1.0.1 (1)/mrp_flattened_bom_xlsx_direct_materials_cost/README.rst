Usage
=====

To use this module, you need to:

#. Go to 'Manufacturing / Products / Bill of Materials'

#. Select a BOM or more BOMS

   *(Could be interesting to modify quantities of these BOMs)*

#. Go to 'Print / Export Flattened BOM to Excel'.
