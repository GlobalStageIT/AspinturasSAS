MercadoLibre Publisher

MercadoLibre Publisher / Connector for Odoo.

What it is?

MercadoLibre Publisher (meli_oerp) by Moldeo Interactive is a MercadoLibre Connector for Odoo.
This is the core module to start importing sales order from one of the top 5 marketplaces.

Features

  • Massive publication from Odoo to MercadoLibre / Import all products from MercadoLibre to Odoo
    • Images included
    • Variations with all attributes combinations
    • SKU options
  • Get Orders, Clients and Shipping information from MercadoLibre to Odoo
    • Import pack orders
    • Import fee cost and shipping cost
    • Use SKU to find existent Product
  • Massive and automatic synchronization
    • Stock and Prices
    • Realtime for orders using mercadolibre notifications
    • Questions
