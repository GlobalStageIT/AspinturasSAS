MercadoLibre Stock Extension

MercadoLibre Stock / MercadoLibre Publisher Extension.

What it is?

MercadoLibre Stock Extension (meli_oerp_stock) by Moldeo Interactive is a MercadoLibre Publisher/Connector (meli_oerp) Extension for Odoo

Features

Choose speficic stock rules for importing sale orders from MercadoLibre and post correct product stock to MercadoLibre.

  • Automatic warehouse selection for Orders
    • Select default warehouse for MercadoLibre.
    • Select specific warehouse for FulFillment Orders.
    • Choose specific warehouses based on logistic type shipment attribute.

  • Automatic location selection for Stock posting
    • Check location as MercadoLibre active location to post.
    • Choose default location or leave blank and system will add all them for posting.
  • Automatic location selection Updating Odoo inventory from ML
    • Usually for first synchronization from ML to Odoo, set the stock quantities in each location accordingly
  • Sku Rules
    • Rule for mapping external Sku to local Sku (ideal for packs)
    • Rule for mapping formula

Configure WareHouse for Orders and Locations for Stock posting to MercadoLibre.
