MercadoLibre Accounting Extension

MercadoLibre Accounting / MercadoLibre Publisher Extension.

What it is?

MercadoLibre Accounting Extension (meli_oerp_accounting) by Moldeo Interactive is a MercadoLibre Publisher/Connector (meli_oerp) Extension for Odoo

Features

Records payments on journal, also fee cost and shipping cost to MercadoPago/MercadoLibre journals.

  • Automatic accounting movement creation
    • Payment records creation based on sale orders from MercadoLibre to Odoo (ready to councile invoices)
    • Fee (comisiones de ML) payment creation (outgoing)
    • Shipping cost

Configure journals and MercadoLibre Partner and set automatic accounting movement registration.
